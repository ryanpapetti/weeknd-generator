Project Summary

The Weeknd Lyrics Generator is a GPT-Neo based Model that uses Natural Language Processing to create Weeknd song lyrics. The model takes a prompt stated by the user and uses its advanced training to produce a result that is guaranteed to wow you. Using lyrics from all of the Weeknd's tracks on Genius, we were able to fabricate the best of the best model for our project.

Collecting The Data

We first collected lyrics from the Genius API through our own client application. We downloaded the lyrics into a text file friendly for the model. 

Training The Model

After collecting the data, we cleaned it, removing all unnecessary punctuation and emojis through regex and the emoji python library. We then trained the GPT-Neo model using the aitextgen library in python, with our cleaned data.
